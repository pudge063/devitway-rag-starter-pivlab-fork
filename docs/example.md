# Пример документа

Это тестовый файл, чтобы `python ingest.py ./docs` отработал сразу после клонирования.

Положи сюда свои `.md`, `.txt`, `.pdf`, `.docx` — можно во вложенных папках
(ридер обходит директории рекурсивно). Затем запусти ingest ещё раз: обновятся
только изменившиеся файлы.

## Что стоит знать про эту связку

- **RAG — это отдельный слой**, а не часть агента. Он векторизует документы и
  отдаёт релевантные фрагменты по запросу.
- **ollama** здесь крутит только эмбеддинг-модель (`bge-m3`) — превращает текст
  в векторы локально.
- **Qdrant** хранит эти векторы и ищет похожие.
- **MCP-сервер** — единая точка входа: к нему цепляются и qwen-code, и Claude Code.

Проверь, что поиск работает: спроси у агента что-нибудь по содержимому этого файла,
например «что делает ollama в этой связке» — он должен позвать инструмент `search`.
